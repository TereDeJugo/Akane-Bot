## Build

First, download and install [NodeJS](https://nodejs.org/en/download/)

Start the node project: 

```
npm init -y
```

Install the packages

```
npm install discord.js megadb dotenv 
```

And you can start using Akane Bot, you just have to create a `.env` file and put the following values:

```.env
Token=Token
Color=Embed_Color
````

If you use Replit, you can go to the [Akane Project](https://replit.com/@TereDeJugo/Akane-Bot), and make it Fork, and you will have everything installed, you only need to make the file `.env` !
